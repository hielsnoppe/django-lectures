HTML Escaping

See also Automatic HTML escaping.

By default, all variables are escaped:

{{ my_html }}

You can turn off escaping for a section of a template:

{% autoescape off %} {{ my_html }} {% endautoescape %}

You can turn off escape for a single variable:

{{ my_html|safe }}

When turned off, you can still manually escape a single variable:

{% autoescape off %} {{ my_html|escape }} {% endautoescape %}
